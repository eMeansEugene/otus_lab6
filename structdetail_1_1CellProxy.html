Detailed Description

template<typename T, T Default>
struct detail::CellProxy< T, Default >

Proxy-объект для доступа к отдельной ячейке матрицы.

Реализует паттерн Proxy (Заместитель): перехватывает операции чтения и записи, позволяя различить их без немедленного обращения к хранилищу.

  • При записи (operator=) вставляет или удаляет элемент из map.
  • При чтении (operator T) возвращает значение или Default.

Записывает значение в ячейку.

Если val равен Default — ячейка освобождается (удаляется из map). Возвращает *this для поддержки цепочек присваивания: ((matrix[0][0] = 5) = 0) = 3.
