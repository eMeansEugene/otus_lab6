Detailed Description

template<typename T, T Default = T{}>
class SparseMatrix< T, Default >

Двумерная разреженная бесконечная матрица.

Хранит только занятые ячейки в std::map. Индексация через цепочку Proxy-объектов позволяет корректно различать чтение и запись: чтение свободной ячейки не создаёт запись в хранилище.

Поддерживает range-based for — итерация по занятым ячейкам возвращает std::tuple<int, int, T> (row, col, value).
