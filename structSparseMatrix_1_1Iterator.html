Detailed Description

template<typename T, T Default = T{}>
struct SparseMatrix< T, Default >::Iterator

Итератор по занятым ячейкам матрицы.

Оборачивает итератор внутреннего std::map и преобразует каждый элемент в std::tuple<int, int, T>.
